自动执行函数

自动执行函数顾名思义就是当JS引擎执行到该函数时自动执行，不需要引用。

声明函数自执行

(function (){
    alert("a")
})();

还有+、！、~等操作符可使声明函数自执行

表达式函数自执行

var b = function(){
    alert("b")
}();

表达式函数只需要在函数最后加入()既可使函数自执行
